# Neuro-Research Archive: Experimental Design & Behavioral Analytics 🧠🔬

This repository is a professional archive of experimental protocols and behavioral scripting developed between 2022 and 2023. It showcases the implementation of high-precision cognitive tasks and psychometric assessments using **PsyToolkit**.

---

##  Projects Included

###  [2023] Visual Attention & Cognitive Bias Protocol
[cite_start]Investigation into the interaction between clinical traits and cognitive processing[cite: 33].
* [cite_start]**Core Paradigms:** **Posner Cueing Task** and **Recognition Memory Task** with millisecond-precision tracking of **Reaction Times (RT)**[cite: 33, 34].
* [cite_start]**Psychometrics:** Integrated scoring for **ECR-12** (Attachment), **STAI-Y2** (Anxiety), **AST** (Ambiguous Scenarios Test), and **LCSQ** (Looming Cognitive Style Questionnaire)[cite: 33].
* [cite_start]**Logic:** Advanced flow control with conditional branching and custom anonymization algorithms[cite: 33, 34].

###  [2022] Food Categorization & Body Image Study
[cite_start]Group research project on how environmental consistency and individual body perception bias object classification.
* [cite_start]**Core Paradigm:** Interactive **"Scaffale" (Shelf) task** using standardized food stimuli (Food-475 and Food101 databases).
* [cite_start]**Psychometrics:** Integrated the **Body Dissatisfaction Scale Assessment** (Mutale et al., 2016) to quantify the gap between "Real Self" and "Ideal Self" via silhouette figures.
* [cite_start]**Biometrics & Demographics:** Automated collection of **BMI (Body Mass Index)**, dietary habits (e.g., ketogenic, Mediterranean), and physical activity levels to correlate with cognitive performance.
* [cite_start]**Experimental Logic:** Managed complex stimuli randomization and analysis of **Reaction Times** in context-dependent environments.

##  How to Use the Code

These projects require the **PsyToolkit** software or the **PsyToolkit platform** ([psytoolkit.org](https://www.psytoolkit.org/)).

1. **Create an Account:** Register on [psytoolkit.org](https://www.psytoolkit.org/).
2. **Upload Files:** Create a new experiment/survey and upload the `.psy` scripts found in this repository.
3. [cite_start]**Manage Assets:** Upload the necessary bitmaps/images mentioned in the script tables (e.g., stimuli for the shelf or memory tasks)[cite: 33, 34].
4. **Compile & Run:** Use the online platform to compile the code. You can run the experiment online or download the HTML file for offline execution.
5. [cite_start]**Integration:** You can create a survey and include these experiments as part of a larger research battery[cite: 33].
